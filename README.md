# tkouter

* [Installation](#installation)
* [Requirements](#requirements)

## Installation

```sh
$ pip install tkouter
```

or you can clone this repo directly.

```sh
$ git clone https://github.com/dokelung/tkouter.git
```

## Requirements

* Python3.5 or later
* [Jinja2](http://jinja.pocoo.org/docs/2.10/)
